# Telegram Temp Mail Bot 📧

Асинхронный Telegram-бот на Python, который работает как сервис временной почты. Бот позволяет создавать случайные email-адреса на вашем домене, принимает входящие письма (включая HTML и вложения), сохраняет их и отправляет уведомление в Telegram со ссылкой на красивый веб-интерфейс для просмотра.

## 🌟 Особенности

* **Генерация ящиков:** Создание случайных email-адресов в один клик.
* **Свой SMTP-сервер:** Прием писем напрямую на 25 порту (без использования сторонних API).
* **Поддержка вложений:** Парсинг и сохранение файлов любых форматов, предпросмотр изображений.
* **Веб-интерфейс:** Встроенный веб-сервер выдает письма в формате HTML со стильным дизайном Material Design 3 (Dark Theme).
* **Автоочистка:** Письма и вложения автоматически удаляются с сервера через 1 час для экономии места.
* **Авторизация:** Доступ к боту защищен паролем.

## 🛠 Требования

* **VPS / Выделенный сервер** с белым IP-адресом.
* **Открытые порты:** `25` (для SMTP) и `80` (для Web). *Примечание: для прослушивания этих портов потребуются права root.*
* **Доменное имя**, привязанное к вашему серверу.
* **Python 3.8+**

## ⚙️ Настройка DNS

Чтобы ваш сервер мог получать почту, необходимо правильно настроить DNS-записи у вашего регистратора домена:

1.  **A-запись:** Направьте ваш домен (например, `yourdomain.com`) на IP-адрес вашего сервера.
2.  **MX-запись:** Создайте MX-запись со значением вашего домена (например, `yourdomain.com`) и приоритетом `10`.

## 🚀 Установка и запуск

### 1. Клонирование и зависимости
Скачайте код и установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### 2. Настройка конфигурации

Откройте скрипт и отредактируйте блок настроек в начале файла:

```python
BOT_TOKEN = "ВАШ_ТОКЕН_БОТА_ОТ_BOTFATHER"
DOMAIN = "yourdomain.com" # Ваш домен
BOT_PASSWORD = "ваш_пароль_для_доступа" 
PROXY_URL = None # Укажите прокси, если требуется (например, для обхода блокировок Telegram)

WEB_PORT = 80
WEB_URL_BASE = "http://yourdomain.com" # Ссылка, которая будет приходить в Telegram

```

### 3. Запуск

Поскольку скрипт использует порты 25 и 80, его необходимо запускать с правами суперпользователя:

```bash
sudo python3 main.py

```

*(Для работы в фоновом режиме рекомендуется использовать `systemd`, `tmux` или `screen`)*.

## 📱 Команды бота

* `/start` — Запуск бота и запрос пароля.
* `/create` — Создать новый временный email-адрес (формат `[random8]@domain.com`).
* `/panel` — Посмотреть список всех созданных ящиков текущего пользователя.

Вам нужно отправить боту пароль, указанный в `BOT_PASSWORD`, обычным текстовым сообщением, чтобы авторизоваться и получить доступ к командам.

## 🗄 Структура базы данных

Бот автоматически создает SQLite базу `mailbot.db` со следующими таблицами:

* `users` — Авторизованные пользователи Telegram.
* `emails` — Привязка созданных email-адресов к ID пользователей.
* `messages` — Хранение сгенерированного HTML писем и времени их истечения.

Папка `attachments/` создается автоматически для хранения временных файлов, прикрепленных к письмам.

## ⚠️ Важные замечания по безопасности

1. **Пароль администратора:** Задайте сложный `BOT_PASSWORD`.
2. **Path Traversal:** В коде уже реализована базовая защита при скачивании вложений, но не рекомендуется запускать веб-сервер от имени root на критически важных серверах. Для продакшена лучше использовать Reverse Proxy (например, Nginx), проксируя запросы на порты веб-сервера, запущенного без прав root.
3. **Блокировка 25 порта:** Многие хостинг-провайдеры (Hetzner, DigitalOcean, AWS и др.) по умолчанию блокируют входящий и исходящий трафик на 25 порту для борьбы со спамом. Возможно, вам потребуется написать в поддержку хостинга для его разблокировки.
